# MindPulse AI - Mobile App

The MindPulse AI mobile application is a cross-platform emotional wellness and biometric health companion built using Flutter.

## Features

- AI-powered emotional health assistant
- Mood tracking and analysis
- Heart rate and wellness insights
- Conversational AI companion
- Stress and sleep analytics
- Health dashboard visualization

## Technology Stack

- Flutter
- Dart
- Firebase
- HealthKit
- Google Fit
- OpenAI / Open-source LLMs

## Future Integrations

- Wearable devices
- Biometric sensors
- Voice emotion analysis
- Predictive wellness AI

## Status

Initial MVP development in progress.
